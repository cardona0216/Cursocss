Flex Box

Es un modelo de diseño en CSS que se utiliza para crear diseños más eficientes y flexibles dentro de un contenedor. Con Flexbox, puedes organizar, alinear y distribuir elementos dentro de un contenedor, permitiendo un diseño más dinámico y adaptable a diferentes tamaños de pantalla y dispositivos.
