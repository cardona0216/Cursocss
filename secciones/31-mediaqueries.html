IMAGEN

Las "media queries" en CSS son una característica que permite adaptar el diseño de una página web a diferentes dispositivos y tamaños de pantalla. Con las media queries, puedes definir reglas de estilo específicas que se aplicarán según ciertos criterios, como el ancho de la pantalla, la orientación del dispositivo, la resolución, entre otros. Por ejemplo, puedes utilizar media queries para establecer estilos diferentes para dispositivos móviles, tabletas y computadoras de escritorio. Esto te permite crear diseños responsivos que se ajusten automáticamente a las necesidades de cada dispositivo, mejorando así la experiencia del usuario.
