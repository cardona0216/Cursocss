Pseudoclases

Una pseudoclase en CSS es un selector que permite seleccionar elementos en función de su estado o posición dentro del árbol de documentos HTML. Se utilizan para
aplicar estilos a elementos basados en interacciones del usuario o en el estado del elemento en sí mismo.

litado de algunas Pseudoclases

:active: Aplica estilos a un elemento mientras está siendo activado por el usuario (por ejemplo, al hacer clic).

:checked: Selecciona elementos de entrada (input) que están marcados o seleccionados.

:default: Selecciona el elemento preseleccionado o predeterminado en un grupo.

:disabled: Selecciona elementos deshabilitados, como campos de entrada o botones.

:empty: Selecciona elementos que no tienen hijos.

:enabled: Selecciona elementos habilitados, como campos de entrada o botones.

:first-child: Selecciona el primer hijo de un elemento padre.

:first-of-type: Selecciona el primer elemento de un tipo específico dentro de su contenedor.

:focus: Aplica estilos a un elemento cuando recibe el foco de atención, como al hacer clic en un campo de entrada.

:focus-within: Aplica estilos a un elemento si algún elemento secundario tiene el foco.

:hover: Establece estilos cuando el cursor del mouse se encuentra sobre un elemento.

:invalid: Selecciona elementos de entrada que contienen datos inválidos.

:lang(): Selecciona elementos basados en el idioma especificado.

:last-child: Selecciona el último hijo de un elemento padre.

:last-of-type: Selecciona el último elemento de un tipo específico dentro de su contenedor.

:link: Selecciona enlaces que no han sido visitados.

:not(): Selecciona elementos que no coinciden con un selector específico.

:nth-child(): Selecciona elementos que son hijos específicos de su padre basados en una fórmula.

:nth-of-type(): Selecciona elementos basados en su posición como hijo de un tipo específico dentro de su contenedor.

:only-child: Selecciona elementos que son el único hijo de su padre.

:optional: Selecciona elementos de entrada que son opcionales.

:read-only: Selecciona elementos de entrada que son de solo lectura.

:read-write: Selecciona elementos de entrada que son de lectura/escritura.

:required: Selecciona elementos de entrada que son obligatorios.

:root: Selecciona el elemento raíz del documento (normalmente el elemento ).

:target: Selecciona el elemento anclado actualmente en el URL.

:valid: Selecciona elementos de entrada que contienen datos válidos.

:visited: Selecciona enlaces que han sido visitados.

:before: Permite agregar contenido antes del contenido del elemento seleccionado.

:after: Permite agregar contenido después del contenido del elemento seleccionado.
